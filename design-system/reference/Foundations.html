04

Principles

How the system behaves when the rules aren't written down.

A

Editorial, not decorated

Whitespace, hairline rules, and a strict grid do the work. Sharp corners, quiet shadows, no gradients. Let the imagery and the type carry the beauty.

B

Two audiences, one voice

Cool and fashion-forward for talent; precise and professional for business. The Didone flatters the models, the grotesque reassures the bookers.

C

Cobalt with intent

The accent is a scalpel, not a highlighter. One cobalt mark per view — a booking action, a live badge, a single word set apart. Red is reserved for danger only.
